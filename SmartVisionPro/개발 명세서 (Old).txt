Vision 검사 프로그램을 클라이언트로 개발 명세서

프로젝트 목표: 카메라를 연결하고 이미지를 획득하여 품질 검사하는 Vision 클라이언트 프로그램 개발.
Vision 검사 프로그램을 클라이언트로 개발하려고 해.
프로젝트는 Visual studio 2022, Windows Forms 앱(.NET Framework), 생성 했어요.
카메라는 카메라 제조사에 따라 여러 방식으로 연결할거에요 (제조사별 프레임그래버는 별도 제공)
조명은 별도 사용하고 시리얼 통신으로 연결 할거에요.
카메라에서 취득한 이미지는 Opencv 4.8.0 라이브러리를 사용하여 이미지 프로세싱을 진행하고 결과는 상위 시스템으로 전달하기 위해 비동기 소켓 통신 적용 필요해요.

솔루션: SmartVisionPro
실행 프로젝트: VisionWorks

F-01 (Lib_Core 라이브러리 프로젝트 구조설계)
- 싱글턴 패턴을 사용해야하며 Lib_Core 라이브러리 프로젝트의 Singleton.cs 파일을 사용합니다.
- Singleton 클래스를 상속한 Core 클래스를 만들어 주세요.
- Core 클래스는 초기화 & 종료 코드 설정해줘
- 모든 매니저 클래스는 Singleton 클래스를 상속하여 만들고 Core 클래스에서 초기화 및 종료해야 합니다.
- Thread 세이프 로직 적용해야 합니다.

F-02 (UI 설계)
이미지 뷰 만들기
카메라 버퍼 이미지 또는 로컬 저장된 이미지를 로딩하여 표시할 수 있는 픽쳐박스로 만들어줘 (바탕은 블랙)
로딩 가능한 이미지 포멧은 bmp, jpg, png로 설정해줘.
이미지 끈김 현상 개선을 위해 더불 버퍼링 적용 검토 해줘.

F-03 (UI 설계)
이미지 분석을 위한 줌 기능 구현
선택하여 로딩한 이미지에 대해 확대 (16배), 축소 (1배), 이미지 이동 (팬기능), 이미지 원위치 (로딩했던 배율로 원위치)
모든 동작은 마우스 이벤트로 구현해줘.
확대/축소는 마우스 휠 사용 (앞/뒤)
이미지 이동은 마우스 왼쪽 버튼을 누른 상태에서 팬기능 적용
이미지 원위치는 마우스 오른쪽 버튼 클릭 시 팝업 메뉴로 선택	Must-Have	UI 구성 (CMFCApplicationView)

F-04	(UI 설계)
메뉴 구성
이미지 파일 선택하여 열기
종료	Must-Have	UI 구성
