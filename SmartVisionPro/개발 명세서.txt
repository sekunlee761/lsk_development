Vision 검사 프로그램을 클라이언트로 개발 명세서

프로젝트 목표: 카메라를 연결하고 이미지를 획득하여 품질 검사하는 Vision 클라이언트 프로그램 개발.
Vision 검사 프로그램을 클라이언트로 개발하려고 해.
프로젝트는 Visual studio 2022, Windows Forms 앱(.NET Framework), 생성 했어요.
카메라는 카메라 제조사에 따라 여러 방식으로 연결할거에요 (제조사별 프레임그래버는 별도 제공)
조명은 별도 사용하고 시리얼 통신으로 연결 할거에요.
카메라에서 취득한 이미지는 Opencv 4.8.0 라이브러리를 사용하여 이미지 프로세싱을 진행하고 결과는 상위 시스템으로 전달하기 위해 비동기 소켓 통신 적용 필요해요.

솔루션: SmartVisionPro
실행 프로젝트: VisionWorks

모든 메니저는 자동 등록/종료 로직을 추가해 전체 초기화 흐름에 포함

F-01 (Lib_Core 라이브러리 프로젝트 구조설계)
- 싱글턴 패턴을 사용해야하며 Lib_Core 라이브러리 프로젝트의 Singleton.cs 파일을 사용합니다.
- Singleton 클래스를 상속한 Core 클래스를 만들어 주세요.
- Core 클래스는 초기화 & 종료 코드 설정해줘
- 모든 매니저 클래스는 Singleton 클래스를 상속하여 만들고 Core 클래스에서 초기화 및 종료해야 합니다.
- Thread 세이프 로직 적용해야 합니다.

F-02 (Lib_Core 라이브러리 프로젝트 구조설계)
- 경로를 관리하는 PathManager 만들어줘.

F-03 (Lib_Core 라이브러리 프로젝트 구조설계)
- 다양한 형식의 파일을 로딩, 수정, 저장, 삭제하는 파일를 관리하는 FileManager 만들어줘.
- 우선 json, txt, xlsx, bmp, png 등 입니다.

F-04 (Lib_Core 라이브러리 프로젝트 구조설계)
- 시퀀스 설계에서 사용할 타이머 관리자 설계해줘. 다형형을 적용하고 베이스 클래스를 상속 받은 타이머 클래스로 만들어줘. 타이머 동작은 고속, 정밀 타이머, loop 타이머, off 타이머, on 타이머로 우선 설계해줘.
각 워커 스래드에서 사용할 수 있도록 스래드 세이프하게 설계해줘.

F-05 (Lib_Core 라이브러리 프로젝트 구조설계)
- 프로젝트에서 레시피 효율적으로 관리할수 있는 레시피 관리자 만들어줘. 다형형을 적용하고 베이스 클래스를 상속 받은 레시피 클래스로 만들어줘.
- 활성 레시피, 라스트 레시피 구조 설계해줘.

F-06 (Lib_Core 라이브러리 프로젝트 구조설계)
- 클라이언트 프로젝트에서 제품 정보를 효율적으로 관리할수 있는 제품 관리자 만들어줘. 다형형을 적용하고 베이스 클래스를 상속 받은 제품 클래스로 만들어줘.
- Substrate, Wafer 등 행, 열로 구성되어 있어요.
